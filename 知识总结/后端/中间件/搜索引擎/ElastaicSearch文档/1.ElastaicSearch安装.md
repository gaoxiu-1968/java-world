# ElastaticSearch安装

> ElastaicSearch 依赖java环境

## windows安装

### 下载

下载地址：https://www.elastic.co/cn/downloads/elasticsearch

### 目录介绍

```
bin 启动文件
config 配置文件
	elasticsearch.yml
		es配置文件：端口，集群，跨域
	jvm.options
		jvm配置：存储大小
	log4j2.properties
		日志配置
data 数据存储
jdk jdk
lib 依赖jar
logs 日志
modules 功能模块
plugins 插件
	ki分词器插件

```

### 启动

双击elasticsearch.bat

### 安装可视化工具elasticsearch-head

> elasticsearch-head依赖nodejs

#### 下载

下载地址：https://github.com/mobz/elasticsearch-head/tags

#### 安装

```
1.
npm install
2.
npm run start
```

#### 解决连接ElastaicSearch引擎跨域问题

```
elasticsearch.yml文件中

http.cors.enabled: true  开启cors
http.cors.allow-origin: "*"  允许所有跨域请求
```



* head 推荐用查看信息，具体CRUD推荐使用kibana（有提示，而且js语句是格式化的）
